# 🗨️ BakBak Chat

A full-stack, real-time, End-to-End Encrypted chat application built with the MERN stack + Socket.io + WebRTC.

## Tech Stack
- **Backend**: Node.js, Express.js, MongoDB Atlas, Socket.io
- **Frontend**: React (Vite), Tailwind CSS, Socket.io-client, WebRTC
- **Security**: JWT Authentication, Web Crypto API (E2EE)
- **Media**: Cloudinary / Local uploads

## Project Structure
```
BakBak_chat/
├── backend/          # Express API + Socket.io server
├── frontend/         # React (Vite) client app
├── .gitignore
└── README.md
```
